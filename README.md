# 証明写真メーカー

ブラウザ上で AI が背景を自動除去し、各種証明写真として書き出せる Web アプリ。

**すべての処理はブラウザ内で完結し、写真はサーバーや外部 API に送信されません。**

## 主な機能

- **AI 背景除去**: [`@imgly/background-removal`](https://github.com/imgly/background-removal-js)（`isnet` モデル）をブラウザ内で実行
- **12種のサイズプリセット**: 履歴書 / マイナンバーカード（オンライン・印刷）/ 各種パスポート / 免許 / 医学・司法系試験 / 英検 S-CBT / 雇用保険 / 国際運転免許 / TOEIC 等
- **12色の背景**: スタンダードブルー、ホワイト、レッド、ピンク、オレンジ、グリーン、グレー系、チャコール、ブラック、ブルー、イエロー
- **単色 / 上→白グラデーション**を切替
- **直感的な操作**: ドラッグで位置調整、マウスホイール / ピンチで拡大縮小、左右反転
- **書き出し解像度プリセット**:
  - 印刷標準 (300 dpi)
  - プロ印刷 (350 dpi)
  - オンライン申請用 (長辺 640 px 目安)
- レスポンシブ UI（スマホ最適化）

## 開発

```sh
pnpm install   # または npm install
pnpm dev
```

- 開発サーバーは `Cross-Origin-Opener-Policy` / `Cross-Origin-Embedder-Policy` ヘッダを送信し、WebAssembly のマルチスレッド実行（`crossOriginIsolated`）を有効化します（`vite.config.ts`）。
- AI モデル（数十 MB）は初回アクセス時に CDN からブラウザへダウンロードされ、以降はブラウザキャッシュから読み込まれます。**写真そのものは送信されません。**

```sh
pnpm build     # ./dist に静的ビルド
pnpm preview   # ビルド成果物をローカル確認
```

## プライバシー

- アップロードされた画像、推論結果、編集結果のいずれもサーバー側に送信・保存されません。
- 通信が発生するのは、AI モデル（重み）の CDN 取得のみです。
- 本番環境にデプロイする際も、上記2つの HTTP レスポンスヘッダを付与してください（推奨: Cloudflare Pages `_headers` / Vercel `vercel.json`）。

例 (`public/_headers`):
```
/*
  Cross-Origin-Opener-Policy: same-origin
  Cross-Origin-Embedder-Policy: require-corp
```
